# Chemical Engineering Process Simulator

Software used to simulate the material and energy balances of chemical process plants. Applications for this include design studies, engineering studies, design audits, debottlenecking studies, control system check-out, process simulation, dynamic simulation, operator training simulators, pipeline management systems, production management systems, digital twins. Multimaterial 3D Printing with Conventional Powder Metallurgy and Selective Laser Sintering. Simulating Composites Manufacturing Processes.
